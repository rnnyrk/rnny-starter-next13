# rnnyrk starter

Starter for NextJS 14 with:

- Drizzle with SQLite
- React Hook Form
- React Query (React Server Actions)
- Tailwind
- TypeScript
- Zod
- Zustand

## Getting Started

```bash
git clone git@github.com:rnnyrk/rnnyrk-starter.git PROJECT_NAME
cd PROJECT_NAME && pnpm i
pnpm run db:push && pnpm dev
```
